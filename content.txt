==== À propos ====

=== Textuel ===

Phiroom est une plateforme web de gestion d'images pour les photographes.
Ils se compose de deux parties :
    - une partie publique ;
    - une partie à accès restreint.

La partie publique sert à diffuser son travail sous forme :
    - de galeries :
        - des groupes de photos associés à une description publiées par ordre chronologique.
    - d'images du jour :
        - une image mise en avant chaque jour sur une page dédiée.
    - de portfolios :
        - une sélection de galeries mises en avant sur une page dédiée.
    - d'articles :
        - des publication de toutes sortes dont le contenu textuel prédomine.
        - rédaction en markdown ;
        - détails typographiques.

La partie privée sert à :
    - classer vos images :
        - par dossiers ;
        - par collections ;
        - par mots clés ;
    - effectuer des recherches dans vos images :
        - par dates ;
        - par mots clés ;
        - suivant les métadonnées exifs ;
        - suivant les métadonnées iptc ;
    - donner un accès à certaines galeries à des utilisateurs particuliers ;
    - publier du contenu sur la partie publique ;
    - gérer certaines options de configuration du site.

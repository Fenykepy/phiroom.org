About Phiroom

Generality

Phiroom is a web pictures management software, built for photographers.

It's made of :

  • a public part ;
  • a restricted acces part.

Public part

Public part allows you to publish your work under :

  • galleries : groups of pictures associated with a description;
  • picture of the day : one picture per day on top of a specific page;
  • portfolios : some galleries on top of a specific page;
  • articles : publications of any kind where text is dominant.

Descriptions of galleries, portfolios, pictures of day and articles are written in markdown.
Each publication appears on weblog page by chronologic order.

Public part of a phiroom powered website is available here.

Restricted access part

Restricted access part actually allows all public part's administration's tasks, and management of pictures:

  • creating and updating galleries;
  • creating and updating portfolios;
  • creating and updating pictures of day;
  • creating and updating articles;
  • sorting publications by categories and keywords ;
  • importing pictures to the library;
  • sorting pictures by folders;
  • sorting pictures by collections ;
  • changing cms' settings.

In future versions, it should allow restricted access to collections and collections' ensembles to authenticated users with right permissions, and management of pictures' exif and iptc metadatas.

Technical

Phiroom's server part is written in python 3, and use django framework.

Client's part is written in html5, css3, javascript and use jquery framework for the lightbox and some visual effects.

A point of honor is made to keep the whole website accessible to no javascript browsers and screen readers through links of avoidance. However, private part usage without javascript comes with some limitations, and some confort elements of public part (including lightbox) won't be available.

Download

Still in test phase, Phiroom will be available soon, through download of an archive to install, or creation of an account on phiroom.com.

It is nonetheless already possible to get sources from github.
You need to install git executable on your system and run: 

git clone https://github.com/Fenykepy/phiroom.git

Then follow installation instructions in INSTALL.md.
If you wish to take part in the project, please send me a mail (read INSTALL.txt in sources to get it).
